# 🔤🎮 Word Guess Game

A fun and interactive browser-based word guessing game where players reveal a hidden word one letter at a time. Test your vocabulary, choose from different categories, and try to guess the secret word before running out of hearts!

## 🎮 How to Play

1. Click **Start Game**.
2. Select a category.
3. Guess letters one at a time.
4. Correct letters reveal their positions in the word.
5. Incorrect guesses cost one ❤️ heart.
6. You start with **6 hearts**.
7. Guess the entire word before your hearts run out.
8. Win by revealing all letters or try again if you lose.
